## 工厂设计

### 硬光

- 光栅板 （3x3） 高级太阳能板，通过光捕获进行发电。可在发电的同时产出少量硬光。
- 热捕获器 （2x2）特种燃烧发电机。将热能转换为能量的同时，捕获热能转换的光能为硬光以提高能量利用效率。
- 辐射捕获器 （3x3）特种放射性反应堆。将未利用的核能转换为硬光，大幅提升了材料利用效率。
- 还有更多相关的一些内容，合理即可

### 钛

- 钛矿簇：裸露于地表的大块纯矿。可以较高的效率直接进行开采。
- 钛地矿：稀疏程度不一的地表矿脉。
- 铍聚变：由铍吸收来自xen的高能粒子，聚变为钛元素（神人到一定程度了这个）
  - 参考配方：2铍 + 6xen -> 1钛

### 芯片

- 芯片刻印机（2x2）：利用高能激光刻印高纯硅，产出芯片。（2硅 -> 1芯片 2s）
- 芯片打印机（2x4）：利用量子体参与的3D打印技术进行芯片生产，可更高效的产出芯片。（2硅 + 6量子体 -> 2芯片， 1s）
- 光刻机（4x4）：利用化工业，硅和布等物料进行混合制造。可批量生产芯片与量子芯片。（4硅+2合金+12化工 -> 6芯片+2量子芯片）

### 光子流

- 光整流器（2x2）：利用硅对光进行整流。可产生密集光子流用于特殊生产。（1硅 -> 3光子流/s）
- 相位整流器（3x3）：利用相位技术对光子进行整流。产生的光子流更为致密。（1布 -> 12光子流/s）

### 冷却液

- 过冷却器（2x2）：高级冷却液混合器，拥有高级的生产效率。（12/s的冷却液）（比原版那个消耗钛少一点）
- 超冷却器（2x4）：利用肼参与冷却液制造，效率更高。（30/s）

### xen

- 粒子升能器（2x4）：燃烧石墨为等离子，再升能为高能粒子Xen。（3石墨 -> 6xen）
- 等离子浆升能器（2x6）：利用钍将等离子浆进行解离以更为高效的生产高能离子Xen。（2石墨+1钍 -> 15xen）

### 重混合钢

- 冲压工厂（2x2）：将石墨和钛冲压为高硬度的重混合钢。（2钛+1石墨 -> 2重混 2s）
- 重冲压厂（2x4）：在冲压工艺的基础上导入中子流，大幅提高了冲压产能。（3钛+2石墨+6中子流 -> 5重混 2s）

### 塑钢

- 塑化机 （3x3）：同时利用光子流和Xen对钛进行塑化处理。（3钛 -> 5塑钢 2s）

### 钨

- 羟金再构仪：利用质子流参与富集工艺，生产效率更高 （18氨 + 6质子流 -> 3羟，2s）
